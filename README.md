# Portfolio-activities
The completed portfolio activities from Google's Cybersecurity Professional Certificate.

***Topics covered:***

**1. Foundations of Cybersecurity**
- Core concepts of the CIA Traid (Confidentiality, Integrity, Availability)
- Foudnational cybersecurity ethics, compliance, and history
  
**2. Security Risks and Management**
- Threat modeling and risk assessment
- Security frameworks and controls (e.g., NIST CSF, NIST RMF)
- Asset classification, authentication, and cryptography principles
  
**3. Networks and Network Security**
- Network architecture, protocols (TCP/IP), and packet analysis
- Defensive tools and configurations, including firewalls, VPNs, and system hardening
- Practice using networking monitoring tools such as Wireshark and tcpdump

**4. Linux and SQL**
- Linux operating system fundamentals and command-line navigation
- Bash scripting basics for adminstrative tasks
- SQL fundamentals to query databases, extract information, and identify security anomalies

**5. Assets, Threats, and Vulnerabilities**
- Identifying attack vectors, malware, and social engineering (e.g., phishing)
- Vulnerability management, intrustion detection, and endpoint detection/response (EDR)
- Working with Intrustion Detection Systems (IDS) like Suricata

**6. Detection and Response**
- Security Information and Event Management (SIEM) concepts and implementation
- Practice with industry tools like Splunk and Google Chronicle
- Incident management playbooks, containment, eradication, and post-incident activities

**7. Automation with Python**
- Basic Python programming for cybersecurity
- Using Python to automate manual security tasks, analyze logs, and parse data


***Overview:***

**Programming and Query Languages**: Python, SQL, Bash\
**Security Tools and Software**: Wireshark, tcpdump, Splunk, Chronicle, Suricata, Linux Terminal/CLI, Bash shell, Syslog, Auth logs, Firewall logs, Apache access logs\
**Networks, Protocols and Exploits**: TCP/IP, OSI Model, ICMP, UDP, DNS, DHCP, ARP, HTTP, HTTPS, SSL/TLS handshakes, Digital Certificates, VPNs, Proxies, Subnet masks, CIDR notation, DDos, MitM, Port scanning, ARP spoofing, SQLi, XSS\
**Frameworks, Compliance and Access Architecture**: NIST CSF, NIST RMF, OWASP Top 10, CVE, CVSS, RBAC, PoLP, MFA, PKI, AES, RSA, Hashing (SHA-256, MD5), CIA Triad
